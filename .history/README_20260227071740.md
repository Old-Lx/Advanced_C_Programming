# EP5801 Programación Avanzada en C
## Repositorio para la electiva EP5801 de la Universidad Simón Bolívar

### Tarea 1: 
- [x] Crear un main.c que ejecute un "Hello World!" y un calculadora.c que ejecute una calculadora sencilla
- [x] Implementar wildcards en el Makefile (opcional)

### Tarea 2:
- [x] Hacer un programa de assembly que imprima una string que envíe un usuario.

### Tarea 3:
- [x] Implementar el sistema de lectura serial de sensores mediante struct, unions y bitfields

### Tarea 4:
- [x] Memory allocators linear and stack
    1. - [x] Reemplazar la memoria dinámica por memoria estática en los alojadores
    2. - [x] Utilizar alojador de stack para construir una calculadora de stack usando ReversePolishNotation (RPN)

### Tarea 5
- [ ] Usar los vectores para implementar un tipo de data custom para una partícula (posición $(x, y)$)

Debe ser capaz de sumar colecciones de partículas o partículas individuales.

Cada partícula debe tener una función que imprima sus coordenadas.
